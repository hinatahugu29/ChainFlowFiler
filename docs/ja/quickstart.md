# クイックスタートガイド

Chain Flow Filer の基本的な操作概念とキーバインドを紹介します。
マウスでも操作可能ですが、キーボードショートカットを活用することで真価を発揮します。

## 画面構成

画面は大きく分けて3つのエリアで構成されています。

1.  **Sidebar (左)**: よく使うフォルダ(Standard)、お気に入り(Favorites)、ドライブ(Drives)へのアクセス。
2.  **Flow Area (中央〜右)**: ファイル操作を行うメインエリア。左から右へ階層が連なります。
3.  **Address Bar (上)**: 現在地のパス表示と直接入力。

## 基本操作フロー

1.  **移動**: 矢印キー `↑` `↓` でファイルを選択、`Enter` または `→` でフォルダに入ります。
2.  **戻る**: `Backspace` または `Q` キーで親フォルダに戻ります。
3.  **プレビュー**: ファイルを選択して `Space` キーを押すとプレビューが表示されます。
4.  **ペイン操作**: マウスカーソルを別のペインに移動するだけで、即座に操作対象が切り替わります（Hover Auto-Focus）。

## キーボードショートカット一覧

### ナビゲーション
| キー | アクション | 説明 |
| :--- | :--- | :--- |
| `Space` | Quick Look | ファイルの中身をプレビュー表示します |
| `Q` / `Backspace` | Go Up | 親ディレクトリへ移動します |
| `Ctrl + B` | Toggle Sidebar | サイドバーの表示/非表示を切り替えます |

### ファイル操作
| キー | アクション | 説明 |
| :--- | :--- | :--- |
| `Ctrl + C` | コピー | 選択項目をコピーします（マウスオーバーだけで実行可能） |
| `Ctrl + X` | 切り取り | 選択項目を切り取ります |
| `Ctrl + V` | 貼り付け | クリップボードの内容を現在のフォルダに貼り付けます |
| `F2` | リネーム | 名前の変更 |
| `Del` | 削除 | ファイルの削除 |
| `Alt + Click` | マーク (Bucket) | アイテムを「収集カゴ」に入れます（後述） |

### レイアウト・表示
| キー | アクション | 説明 |
| :--- | :--- | :--- |
| `V` | Vertical Split | 新しいレーンを縦に追加し、複数の作業を並行して行えます |
| `Shift + W` | Close View | 現在アクティブなビュー（分割画面）だけを閉じます |
| `C` | Compact Mode | フォルダツリー表示を簡易化し、多くの情報を表示します |
| `Ctrl + T` | New Tab | 新しいタブを開きます |
| `Ctrl + W` | Close Tab | 現在のタブを閉じます |
| `Shift + Scroll` | Resize Lane | [サイドバー上] セクションの高さを調整します |

## 独自の操作概念

### 収集カゴ (Collection Bucket)
複数のフォルダに散らばったファイルを一度に移動・コピーしたい場合に便利です。

1.  ファイル上で `Alt + Click` すると、アイテムが赤くハイライトされます。
    *   フォルダを移動しても、このマークは維持されます。
2.  別のフォルダでも `Alt + Click` で追加できます。
3.  最後に任意のファイルをドラッグすると、マークされた全てのファイルが一緒に運ばれます。
4.  `Alt + C` でマークを全解除できます。

### Hover Auto-Focus (ホバーオートフォーカス)
クリック不要の高速操作を実現します。
マウスカーソルをペイン（リスト）の上に置くだけで、そのペインがアクティブになり `Ctrl+C` などのコマンドを受け付けます。
わざわざクリックしてフォーカスを移す必要はありません。
