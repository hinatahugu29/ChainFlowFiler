# Chain Flow Filer v12.0 ドキュメント

**Chain Flow Filer** へようこそ。
これは、キーボード操作主体のパワーユーザー向けWindows用ファイルマネージャーです。
「深く潜る」カラム表示と「広く並べる」マルチウィンドウの利点を融合し、かつてない没入感と効率を提供します。

## 主要機能

*   **Chain Flow Interface**: 親→子→孫へと連なるフローを視覚的に管理。
*   **Keyboard Centric**: ほぼ全ての操作をキーボードだけで完結。
*   **Quick Look**: Spaceキーで瞬時にファイルをプレビュー。
*   **Hover Auto-Focus**: マウスを動かすだけで直感的にフォーカス移動。
*   **Unified Batch Drag**: 異なるフォルダのファイルを一括収集してドラッグ＆ドロップ。
