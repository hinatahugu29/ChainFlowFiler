# インストールと起動

Chain Flow Filer は **ポータブルアプリケーション** として設計されています。
面倒なインストーラーやレジストリ操作は不要です。

## 必要要件

*   **OS**: Windows 10 / 11 (64-bit)
*   Microsoft Office (Word/Excel) または LibreOffice (PDF変換機能を使用する場合)

## インストール手順

1.  配布された zip ファイル (`ChainFlowFiler_v12.zip`) を解凍してください。
2.  任意の場所（デスクトップ、Program Files、USBメモリなど）にフォルダを配置します。

## アプリケーションの起動

フォルダ内の `ChainFlowFiler_v12.exe` をダブルクリックして起動します。

## アンインストール

フォルダごと削除するだけで完了です。レジストリや他のシステムフォルダを汚すことはありません。

### データの保存場所について

設定ファイル (`session.json`, `favorites.json`) は実行ファイルと同じディレクトリに保存されます。
USBメモリに入れて持ち運ぶことで、どのPCでも同じ環境・設定で作業を継続できます。
