# 機能詳細

Chain Flow Filer のユニークな機能を詳しく解説します。

## 1. ナビゲーションサイドバー (v12.0)

サイドバーは3つのセクションに分かれています。

*   **STANDARD**: デスクトップ、ドキュメント、ダウンロードなどのシステム標準フォルダ。
*   **FAVORITES**: ユーザーが登録したお気に入りフォルダ。
*   **DRIVES**: 接続されているドライブ一覧。

### リサイズとレイアウト
各セクションの境界線（スプリッター）はドラッグして調整可能です。
また、マウスホイールでの微調整も可能です。
*   **Shift + Wheel**: セクションの高さを変更。

セクションを折りたたんだ際、下部に発生する余白は自動的に調整され（Spacer機能）、常に上詰めで見やすいレイアウトが維持されます。

## 2. Quick Look (プレビュー)

`Space` キーで呼び出せるプレビュー機能は、単なる閲覧以上の機能を持っています。

*   **コードプレビュー**: Python, JS, Markdown, AHK などのソースコードをシンタックスハイライト付きで表示。
*   **画像プレビュー**: 一般的な画像フォーマットに対応。
*   **Copy Content**: プレビューウィンドウ右上のボタンを押すと、表示されている「中身」をコピーできます。
    *   テキストファイルならテキスト全文。
    *   画像ファイルなら画像データそのものをクリップボードにコピー（SlackやPhotoshopに直接ペースト可能）。

## 3. PDF変換機能

MS Office (Word, Excel) がインストールされている環境、または LibreOffice がある環境では、ドキュメントファイルを右クリックして **PDFへ変換** することができます。
マーク機能と組み合わせることで、複数のドキュメントを一括でPDF化することも可能です。

## 4. コンテキストメニュー拡張

右クリックメニューには、標準のエクスプローラーにはない便利な機能が追加されています。

*   **Create Shortcut**: カレントフォルダにショートカットを作成。
*   **Copy Path Special**:
    *   `Copy Unix Path`: バックスラッシュをスラッシュに置換してコピー（プログラミング等に便利）。
    *   `Copy as "Path"`: 二重引用符で囲んでコピー。

## 5. ホバー操作 (Frictionless Operations)

v12.0 で強化された「摩擦ゼロ」の操作感です。

*   **クリック不要**: マウスを乗せるだけで操作対象として認識されます。
*   **アクティブビューの明確化**: 現在操作可能なリストには左端に青いボーダーが表示されます。
*   **Shift+W (Unstack)**: 階層が深くなりすぎた時、一番右（または下）のビューだけを閉じて整理できます。
