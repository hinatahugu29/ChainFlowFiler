# Installation & Launch

Chain Flow Filer is designed as a **Portable Application**.
No messy installers or registry operations are required.

## Requirements

*   **OS**: Windows 10 / 11 (64-bit)
*   Microsoft Office (Word/Excel) or LibreOffice (required for PDF conversion feature)

## Installation Steps

1.  Unzip the provided archive (`ChainFlowFiler_v12.zip`).
2.  Place the folder anywhere you like (Desktop, Program Files, USB drive, etc.).

## Launching the Application

Double-click `ChainFlowFiler_v12.exe` inside the folder to launch.

## Uninstall

Simply delete the folder. It does not pollute the registry or other system folders.

### About Data Storage

Configuration files (`session.json`, `favorites.json`) are saved in the same directory as the executable.
By carrying it on a USB drive, you can continue your work with the same environment and settings on any PC.
