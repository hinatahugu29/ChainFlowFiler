# Chain Flow Filer v12.0 Documentation

Welcome to **Chain Flow Filer**.
This is a keyboard-centric, high-performance file manager designed for power users on Windows.
It blends the "deep dive" of Miller Columns with the "parallelism" of multi-window workflows to provide unprecedented immersion and efficiency.

## Key Features

*   **Chain Flow Interface**: Visually manage file flows from parent to child to grandchild.
*   **Keyboard Centric**: Perform almost all operations using only the keyboard.
*   **Quick Look**: Instantly preview files with the Space key.
*   **Hover Auto-Focus**: Intuitive focus switching just by moving your mouse.
*   **Unified Batch Drag**: Collect files from different folders and drag & drop them all at once.
